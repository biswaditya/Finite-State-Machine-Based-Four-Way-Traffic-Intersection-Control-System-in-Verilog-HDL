Designed and simulated a Finite State Machine (FSM)-based four-way traffic intersection control system using Verilog HDL. Implemented a multi-state traffic signal sequencing algorithm with configurable timing intervals for red, yellow, and green phases. Developed synchronous state-transition and output-control logic driven by a 300 MHz clock, ensuring safe and coordinated traffic flow between intersecting roads. Verified functional correctness through RTL simulation and timing-based state validation.
